# Deep Learning Demo (BiLSTM)

Folder ini berisi aplikasi antarmuka Gradio khusus untuk menjalankan model Deep Learning (PyTorch BiLSTM + Attention) secara lokal di komputer Anda.

## Cara Menjalankan

1. Buka terminal dan arahkan ke folder utama proyek:
   ```bash
   cd path/to/sentiment-analysis
   ```

2. (Opsional) Gunakan Virtual Environment.

3. Instal dependensi khusus untuk DL Demo:
   ```bash
   pip install -r app/dl_demo/requirements.txt
   ```

4. Jalankan aplikasi:
   ```bash
   python app/dl_demo/app.py
   ```

5. Buka tautan lokal (biasanya `http://127.0.0.1:7860`) yang muncul di terminal Anda pada browser.
